This program is ment to act as a FlashCard for learning. 
The code works in a way that wwgiven a sentence, example:

Pie - Essential for life on earth. 

The program will given the location of the file create a GUI displaying a "card" with

'Pie' displayed on the front side of the card. 'Essential for life on earth' on the
backside of the card.

NOTE! The program is intended to work on a .docx file.

The program uses blankspaces to distinguish between sentences.

The character '-' is used to devide the sentence into two parts.
